#### Description

Project and Lib android.  

#### Usage:

> 1 - Add it in your root build.gradle at the end of repositories:
> ```
> allprojects {
>    repositories {
>      ...
>      maven { url 'https://jitpack.io' }
>    }
> }
> ```
> 2 - Add the dependency 
> ```
> dependencies {
>   compile 'com.github.JefersonR:Dialog-Custom:2.0.1'
>}
> ```
